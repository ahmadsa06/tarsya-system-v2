# ترسية — منصة تجهيز عروض منافسات اعتماد

> واجهة SaaS عربية (RTL) تساعد الشركات على تحليل كرّاسات منافسات منصة **اعتماد** الحكومية، وتنظيم العروض الفنية والمالية، وتصدير حزمة جاهزة — بجودة عربية عالية وعزل أمني بالتصميم.

**Tarsya** — an Arabic-first (RTL) SaaS interface for preparing technical & financial bids for Saudi government tenders. Built with Next.js 16, React 19, and Tailwind CSS v4.

🔗 **عرض حيّ:** [system.tarsya.net](https://system.tarsya.net)

---

## نظرة عامة

مشروع واجهة (front-end) متكامل يعرض رحلة تجهيز العرض من رفع الكرّاسة حتى التصدير، بثلاث بوابات (الشركة، المالك، الدخول). البيانات الحالية **وهمية للعرض**، وميزة التوليد بالذكاء الاصطناعي معروضة كتجربة بصرية (تُربط بمحرّك فعلي لاحقاً).

> منتج مستقل، غير تابع لمنصة اعتماد الحكومية.

## أبرز المزايا

- **مصفوفة التجهيز** — قلب المنتج: كل متطلب بحالته وفئته وأولويته ومسؤوله ومصدره ودرجة ثقته.
- **تحليل بالذكاء الاصطناعي** — عرض النص المُستخرَج من الكرّاسة مع رقم الصفحة ودرجة الثقة (لا يقترح أسعاراً إطلاقاً).
- **مساحة عمل ثلاثية الأعمدة** — شجرة الأقسام · المحرّر · المساعد الذكي.
- **فحص الجاهزية والتصدير** — كشف النواقص والتناقضات قبل التقديم.
- **بوابة المالك** — لوحة التكلفة الداخلية والربحية حسب الباقة.
- **الأمان** — تدفّق تحقّق بخطوتين (2FA)، وإدارة أدوار (RBAC)، وسياسات خصوصية.

## الصفحات

| المسار | الوصف |
|---|---|
| `/` | الموقع التسويقي |
| `/login` | تسجيل الدخول مع التحقّق بخطوتين |
| `/dashboard` | لوحة القيادة |
| `/tenders` · `/tenders/new` | المنافسات ورفع كرّاسة |
| `/tenders/[id]/matrix` | مصفوفة التجهيز |
| `/tenders/[id]/readiness` | فحص الجاهزية |
| `/tenders/[id]/workspace` | مساحة العمل |
| `/settings` · `/admin` | الإعدادات وبوابة المالك |
| `/design` | نظام التصميم |

## المكدّس التقني

- **[Next.js 16](https://nextjs.org)** (App Router) + **React 19** + **TypeScript**
- **[Tailwind CSS v4](https://tailwindcss.com)** (نظام تصميم Material 3 عبر `@theme`)
- **عربي RTL أصيل** + خط IBM Plex Sans Arabic عبر `next/font`
- **تصدير ثابت** (`output: export`) — يُستضاف على أي خادم بلا Node.js

## جودة الهندسة

- **إمكانية وصول (WCAG AA)**: نوافذ حوارية بحصر تركيز وإغلاق بـ Esc، عناصر تحكّم بـ ARIA، تباين ألوان مُتحقَّق.
- **RTL دقيق**: خصائص منطقية، وأدراج باتجاه صحيح.
- **SSG**: تَوليد مسبق لكل المسارات الديناميكية.
- بناء وأنواع وESLint نظيفة بالكامل.

## التشغيل محلياً

```bash
cd web
npm install
npm run dev      # http://localhost:3000
```

## البناء والنشر (تصدير ثابت)

```bash
npm run build    # ينتج مجلد out/ جاهزاً للرفع
```

ارفع محتويات `out/` إلى مجلد النطاق الفرعي على الاستضافة (يحتوي `.htaccess` لإعداد Apache والتخزين المؤقت).
